# Mutual_Fund_Dashboard
1. Project Title
Mutual Fund SIP Investment Dashboard (Excel)

2. Short Description / Purpose
This project is an interactive Excel dashboard designed to track and analyze Systematic Investment Plan (SIP) mutual fund investments.It helps users understand investment performance, distribution across fund types, and overall portfolio growth through visual analytics.
The purpose of this project is to demonstrate data analysis, visualization, and dashboard-building skills using Excel.

3. Tech Stack

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning & Formatting
* Dashboard Design Techniques

4. Data Source
The dataset used in this project is a simulated financial dataset created for learning and demonstration purposes.
It includes investment-related data such as fund category, AMC, invested amount, current value, and yearly SIP contributions.

5. Features / Highlights

* KPI cards showing total investment, current value, and average returns
* Investment distribution by fund type
* Comparison of invested vs current value
* Yearly SIP contribution trend analysis
* Fund manager performance visualization
* Interactive slicers for filtering
* Clean and user-friendly dashboard layout
